## 1. 项目概述

**产品名称**: PromptBox Pro  
**定位**: 多模态提示词分析与生成工具  
**核心价值**: 逆向倒推多模态提示词，创作从模仿开始  
**在线地址**: http://124.156.200.127:3003  
**GitHub**: https://github.com/oxygen02/promptbox

---

## 2. 核心功能

| 功能模块 | 说明 |
|----------|------|
| **文字文档分析** | 上传文档(Word/PDF/TXT)，AI分析提示词维度 |
| **图片视觉分析** | 上传图片，识别视觉提示词元素 |
| **视频结构分析** | 上传视频，分析镜头/节奏/结构（仅分析，无生成） |
| **网页设计分析** | 输入URL，分析网页设计提示词 |
| **提示词生成** | 基于分析结果，生成可复用提示词 |
| **图片生成** | 调用AI生图API生成图片 |

## 7. 防滥用机制

1. **同一IP限领**: 游客积分仅赠送1次
2. **设备指纹**: 识别同一用户，防止重复领取
3. **积分清零规则**:
   - 赠送积分：當月最後一天23:59清零
   - 月租积分：每月1日重新发放
   - 积分包：购买后30天有效

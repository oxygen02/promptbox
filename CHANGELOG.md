# Changelog

All notable changes to this project will be documented in this file.

## [1.0.3] - 2026-03-31

### Added
- 提示词维度标签更新（中英文一致）
- 社交媒体分享功能（微信、微博、小红书 / Facebook、Twitter、Instagram）
- 腾讯云 CodingPlan API 集成
- 后端服务框架

### Changed
- 上传区域布局调整，与维度区域对齐
- 英文维度标签与中文保持一致

### Fixed
- 社交媒体图标使用官方品牌色
- 文件上传处理优化

## [1.0.2] - 2026-03-28

### Added
- Initial release
- Basic prompt analysis features
- Multi-language support (Chinese/English)
